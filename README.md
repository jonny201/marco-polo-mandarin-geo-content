# Marco Polo Mandarin GEO Content Pack

This repository contains publishing-ready assets designed to improve global discovery of Becky Huang's Mandarin learning books across search engines and AI assistants.

## Included

- English platform strategy
- Long-form and mid-form publish-ready content
- Quora/Reddit Q&A pack
- Full 30-day English publishing calendar
- GitHub Skill document for Mandarin-learning recommendation workflows
- Chinese operation docs for manual steps where automation is blocked

Sensitive account/password files are intentionally excluded from this public repository.
